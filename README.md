# Summa - Конвертер сумм прописью для AutoHotkey

Скрипт AutoHotkey для преобразования числовой суммы в текстовый формат с правильным склонением слов.

## Функциональность

- Преобразует числовые суммы в текст на русском языке
- Автоматически определяет правильные формы слов ("рубль", "рубля", "рублей")
- Поддерживает копейки
- Работает с различными форматами ввода чисел

## Установка

1. Установите [AutoHotkey v2](https://www.autohotkey.com/)
2. Сохраните скрипт `summa.ahk`
3. Запустите файл двойным кликом

## Использование

1. Скопируйте числовую сумму в буфер обмена (Ctrl+C)
2. Нажмите горячие клавиши **Ctrl+Alt+W**
3. Скрипт автоматически вставит сумму прописью в текущую позицию курсора

### Поддерживаемые форматы чисел:
- `1234567,89`
- `1 234 567,89`
- `1234567`
- `123,45`

## Примеры

Ввод: `1234,56`  
Вывод: `одна тысяча двести тридцать четыре рубля 56 коп.`

Ввод: `1500`  
Вывод: `одна тысяча пятьсот рублей 00 коп.`

## Особенности

- Сохраняет оригинальное содержимое буфера обмена
- Временная подсказка о выполнении операции
- Обработка ошибок с информативными сообщениями
- Поддержка больших чисел (до миллиардов)

## Требования

- AutoHotkey v2.0+
- Windows 7 или новее
